# How to contribute

I am pleased that you are reading this because we need volunteer developers to move this project forward.

Resource are listed in the website [openwastewater.org](openwastewater.org). Please feel free for testing the tool.
If you like to report errors or suggesting improvements ou can mail to [info@openwastewater.org](mailto:info@openwastewater.org)

# Submitting changes

Plese send a Pull Request to OpenWasteWater with a clear list of what you've done. You can do this viy email as well.

# Coding conventions

Start reading our code and and the OpenModelica turorials and you are gitting an idea of it. We optimise for readability:

We indent using two spaces (soft tabs)  
We put spaces after list items and method parameters ([1, 2, 3], not [1,2,3]) and around operators (x == 1, not x==1) etc.  
This is open source software. Consider the people who will read your code, and make it look nice for them.
